# TechShop - Магазин компьютерной техники

**Данный проект** - это МРП интернет-магазин компьютерной 
техники, построенный на базе React и Django. 
Проект предоставляет полную функциональность для 
онлайн-оформления заказов компьютерной техники, включая каталог 
товаров, корзину покупок, управление заказами, учет 
пользователей и административные инструменты.

## Краткое описание

TechShop - веб-приложение для оформления заказов
компьютерной техники. Система включает:

- **Каталог товаров** с фильтрацией по категориям, производителям и характеристикам
- **Корзину покупок** с выбором какие товары включить в заказ
- **Заказы пользователя** и архив заказов
- **Систему рейтинга и отзывов** для продуктов
- **Поиск товаров** при помощи полнотекстового поиска
- **Систему рекомендаций для продуктов** на основе заказов других пользователей
- **Систему аутентификации по токену** - Django REST Framework Token
- **Управление учетными данными** на странице профиля
- **Разделение административных ролей** которые имеют дополнительные полномочия:
- 1. **Менеджер** - управление заказами пользователей и взаимодействие с товарами
- 2. **Администратор** - управление ролями других пользователей

## Стек технологий

### Фронтенд
- **React 18** - библиотека пользовательского интерфейса
- **React Router** - управление маршрутизацией
- **Axios** - HTTP-клиент для запросов к API
- **SCSS** - препроцессор для стилей
- **TypeScript** - типизация
- **Zustand** - стейт-менеджер
- **React-hook-form** - реализация форм
- **Swiper** - библиотека для реализации слайдеров
- **Material UI** - библиотека, примененная для реализации звезд рейтинга и отзывов

### Бэкенд
- **Django 4.x** - веб-фреймворк
- **Django REST Framework** - создание REST API
- **PostgreSQL** - система управления базами данных
- **Django CORS Headers** - управление междоменными запросами
- **Whitenoise** - статические файлы

## Установка и запуск

### Предварительные требования

- Node.js 16+
- Python 3.8+
- PostgreSQL
- npm или yarn
- Git

### Установка фронтенда

```bash
cd frontend
npm install
```

### Настройка окружения фронтенда

Создайте файл `.env` в корне фронтенда:

```env
VITE_BACKEND_URL=http://localhost:8000/api/
```

### Установка бэкенда

```bash
cd backend
python -m venv venv
\venv\Scripts\Activate
pip install -r requirements.txt
```

### Настройка базы данных

1. Создайте базу данных PostgreSQL

2. Настройте переменные окружения в `.env` или добавьте в `backend/backend/settings.py`:
```python
DATABASES = {
    'default': {
        'ENGINE': 'django.db.backends.postgresql', # Ваш движок для БД 
        'NAME': 'techshop', # Ваше имя БД
        'USER': 'techshop', # Ваш username
        'PASSWORD': 'your_password', # Ваш пароль
        'HOST': 'localhost', # Ваш хост
        'PORT': '5432', # Ваш порт
    }
}
```

### Запуск приложения

Запустить 2 терминала и выполнить команды:

1. Бэкенд в одном терминале:
```bash
cd backend
python manage.py migrate
python manage.py runserver
```

2. Фронтенд в другом терминале:
```bash
cd frontend
npm run dev
```

- `src/store/` - Хранилище состояния (Zustand)
- `src/hooks/` - Кастомные хуки

## Основные возможности

### 1. Каталог товаров
- Просмотр товаров с фильтрацией по:
  - Категориям и подкатегориям
  - Брендам
  - Ценовому диапазону
  - Наличию на складе

### 2. Корзина покупок
- Добавление/удаление товаров
- Подсчет общей стоимости
- Выбор товаров в один заказ
- Переход к оформлению заказа

### 3. Система рейтинга
- Оценки от 1 до 5 звезд
- Автоматический расчет среднего рейтинга
- Отзывы с текстовым комментарием
- Загрузка фото к отзывам

### 4. Управление заказами
- Создание заказа
- Просмотр истории заказов
- Статусы заказов 
- Администраторская панель управления заказами

### 5. Аутентификация
- Регистрация и вход
- Токен DRF для безопасности
- Хранение токена в localStorage
- Возможность смены пароля и личных данных

### 6. Административные функции
- Добавление и редактирование товаров
- Просмотр и обработка заказов
- Управление ролями пользователей

## API

### Базовый URL
```
http://localhost:8000/api/
```

### Основные эндпоинты

**Аутентификация:**
- `POST /login/` - Вход
- `POST /logout/` - Выход
- `POST /check-auth/` - Проверка аутентификации
- `POST /register/` - Регистрация

**Товары:**
- `GET /products/` - Список товаров (пагинация)
- `GET /products/{productId}/` - Детали товара
- `GET /products/?new=true` - Новинки (2 последних товара)
- `GET /products/search/` - Фильтр по категории
- `POST /products/create/` - Создать продукт (только для менеджера)

**Корзина:**
- `GET /cart-items/` - Получить корзину
- `PATCH /cart-items/{productId}/` - Добавить в корзину
- `DELETE /cart-items/{cartItemId}/` - Удалить из корзины

**Заказы:**
- `GET /user-orders/` - Мои заказы
- `POST /user-orders/` - Создать заказ
- `GET /manager-orders/` - Заказы пользователей для менеджера
- `GET /manager-orders/{orderId}/` -

## Администрирование

### Доступ к админ панели
Административный интерфейс доступен по адресу:
```
http://localhost:8000/admin/
```

### Базовый доступ к админ панели

#### В проекте по умолчанию есть учетная запись администратора:
- Login - admin
- Password - password

**ОБЯЗАТЕЛЬНО ЗАМЕНИТЕ ЭТИ ДАННЫЕ В ПРОФИЛЕ ПОЛЬЗОВАТЕЛЯ!**  

#### Вы также можете создать новую запись администратора

```bash
cd backend
python manage.py createsuperuser
```

### Права доступа
- **Пользователь**: просмотр каталога, добавление в корзину, оформление заказов
- **Менеджер**: Управление статусами заказов, редактирование и добавление нового товара
- **Администратор**: управление ролями пользователей, не имеет доступа к редактированию и добавлению товара
