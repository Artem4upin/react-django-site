# Запуск

С директории проекта

## React

cd frontend

### установка зависимостей

npm install

### Запуск

npm run dev


## Django

cd backend

### Создание виртуального окружения

python -m venv venv
### Активация

.\venv\Scripts\Activate

### Установка зависимостей

pip install -r requirements.txt

### Запуск
python manage.py runserver
